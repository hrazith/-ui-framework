Simple Grid System

This is a grid representation of a simple grid system. This grid is uses percentages of div's to format the placing of elements on the page.

A block level element is as wide as the parent it's inside. we can think of it as 100% wide or fractions that add up to 1.

2/3

This column area is 2/3 the side of the entire width or 66.66% wide.

The way this padding works, is that there is a fixed padding of 20 px to the right side of all columns except the last one. In this case, the column that is 2/3 size of the width, has a right padding of 20px. This helps give spacing between other "col-" elemnets you add into your grid

1/3

This column area is 1/3 the side of the width or 33.33% wide.

This "col-" is the last column. As a result, the last "col-" has no right padding.

2/3 (Opt-in Outside Padding)

This column area is 2/3 the side of the entire width or 66.66% wide.

To add outside padding, add left padding to the grid parent (and optionally top and bottom padding). This 2/3 column has a right padding of 20px, and top and bottom padding of 20px.

1/3

This column area is 1/3 the side of the entire width or 33.33% wide.

In the "col-" that is the last column, restore the right padding. Since the last column is the 1/3 column, it has a right padding of 20px.

You can use any fractions you want, just make sure the column fractions add up to 1
