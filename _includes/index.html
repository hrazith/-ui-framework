Consult Works Sandbox


Welcome to the Consult Works Sandbox. Here we have common elements, collections, modules and more, that are used accross the Consult Works applications.

Our goal was to create a place where designers and devlopers can go to for their application creation without take styles from an old application and recustomizing it all over again.

This is a place where the User Experience Team, and Developers can reference these common components and share UI for designing their application.
